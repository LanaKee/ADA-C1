# Engage
## 💡 Big Idea
발전

## Essential Question
포항에서 어떤 발전을 할 수 있을까?

## Challenge Statement
포항에서의 긍정적인 변화의 기반을 마련해주는 앱을 만들자

# Investigate
## Guiding Questions
- 아카데미 사람들은 발전하고 싶어할까? 그렇다면 왜 발전하고 싶어할까?
- 포항에서 접근하기 슁누 자기계발은 무엇이 있을까?
- 이전 기수 러너들이 여기에서 이뤄낸 발전은 무엇이 있을까?
- 포항의 러너들은 어떻게 자기계발을 위해 관리하고 있을까? 그들의 효율적인 방법은 무엇인가?
- 발전하고 싶지 않아하는 사람들이 있을가? 그렇다면 그 이유가 무엇일까?
- 포항으로 오기 전과 후 발전에 어떤 차이가 있는가?

## Insights from Research
1. 포항의 물리적 인프라 부족
- 포항은 교통편이 수도권에 비해 불편하다(적다)
- 포항에서의 문화 활동은 특정분야로 한정된다.
- 긴급한 업무처리가 어렵다.
- 포항은 트렌드에 비교적 둔감하다.
2. 외로움과 적응의 어려움
- 외부숙소를 사용하는 사람들은 심리적 어려움을 겪는다.
- 관련분야가 아닌 분들은 어려움을 겪을 수 있다.
- 아카데미는 일반적인 사회환경과 다른 환경을 가지고 있다.
3. 멘토링 초기 방향 설정 부재
- e.g. 아카데미 활동에서 팀은 본인들의 결정을 굳게 밀고 나간다.
- 아카데미 멘토링은 자율적이다.
- 적절한 기술 수준(아키텍쳐)을 판단할 기준이 없다.
- 아카데미 러너들은 활동 초반에 멘토의 의도를 파악하기 어렵다.
4. 멘토링 과정에서의 감정 소모
- 멘토와 러너 사이에 명확한 선이 없다
- 아카데미 활동중에 함께하는 시간이 많다보니 공과 사가 구분이 어렵다.
- 아카데미 멘토링은 자율적이다.
5. 러너들의 성장 동기, 지속성 문제
- 내적동기가 있어도, 완전 자율주행은 어렵다.
- 외적동기가 성장활동에 큰 영향을 주고 있다.
- 소규모 모임에 지나치게 의존할 수 있다.
- 외적동기에 의존하는 러너들이 많다.
- 성장 동기가 외적동기라면 해당 동기가 실추되었을 때 의지를 잃는다.
6. 커뮤니티 연결 시스템의 부재
- 포항에는 개발자 커뮤니티가 모일 공간이 부족하다. (사람도 없긴 한데...)
- 아카데이 외의 공간에서 만남을 갖기가 어렵다.
7. 옵스팀과 러너의 접점 부족
- 옵스팀의 활동은 러너들에게 가시적이지 않다.
- 옵스팀과 접점이 없다. -> 층이 분리되어있다.
- 옵스팀의 일하는 모습을 보지 못한다.
- 옵스팀은 열심히 일하는 중
- 러너들은 옵스팀이 정확히 무슨일을 하는지 알 수가 없다.

## Challenge Response
> 새로운 시작을 어려워하는 사용자들이 
목표 분할을 통해 시작의 허들을 낮추는 것으로
시작하는 힘과 방식(실행력)을 얻도록 도와주는 앱.
